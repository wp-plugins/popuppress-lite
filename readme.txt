== Description ==

PopupPress is a Wordpress plugin that lets you easily create elegant overlapping windows. PopupPress Popups lets you create any type of content using WYSIWYG content editor.

#### PopupPress can be used to display:
* Advertising banners
* Contact forms
* Notifications
* Alert messages
* Product description
* Work portfolio
* Product images
* Google maps
* Notices
* Or other information

#### Main Features

* You can add **any content type** such as images, videos, shortcodes, etc
* Compatibility with Gravity Forms, Contact Form 7 and much more
* **Auto Open Popups**, Opens automatically on page load
* Show popup on Home, specific posts, pages, post types, etc
* Position Type (Absolute or Fixed). X, Y Position
* Popup Width and Popup Height
* Open and close popup after X seconds
* Show popup to logged or non logged users
* **Overlay color** , change background color Overlay
* Show / Hide title popup
* Options to customize the button Popup
* Responsive Popups
* Disable popups on mobile devices
* **Configuration page** to customize the Plugin Options.
* and more!

== Installation ==

1. Unzip and Upload the directory 'popup-press-lite' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to PopupPress Lite menu and add as many popups as needed
